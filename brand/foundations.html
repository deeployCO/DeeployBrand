10 — Pixel license

One ornament, used well

The logo's descending-square motif is the only decorative element the brand system allows itself. One appearance per page is the ceiling.

In empty states

Top of empty panels. Pulses softly, then rests. Never used alongside the inline loader.

As inline loader
Syncing campaign data…

Replaces a spinner inline. 4 squares, staggered 100ms, one bounce up.

As section rule
Section break

Only when a divider needs semantic weight. Not a substitute for a plain <hr>.

Cascade animation

Used in brand moments — hero intros, loading screens, section transitions.

One appearance per page is the ceiling. If the pixel mark is already in the empty state, don't also use the inline loader on the same view. Restraint is why it reads as intentional rather than template-filler.
